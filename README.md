# ERP System - Web Application with Angular and Firebase

This is a highly interactive **Enterprise Resource Planning (ERP)** web application designed to manage various business processes in an efficient and centralized manner. Built with **Angular** and powered by **Firebase**, it integrates real-time data handling and authentication to streamline operations for small to medium-sized businesses.

## Key Features:
- Modular structure for managing inventory, sales, finances, and more.
- Real-time database synchronization via **Firebase Firestore**.
- User authentication and role management with **Firebase Authentication**.
- Optimized for performance with **Angular** and designed as a **Single Page Application (SPA)** for seamless user interaction.
- Hosted and deployed using **Firebase Hosting** with GitHub Actions for continuous integration.

## Technologies Used:
- **Angular** for the front-end interface and client-side routing.
- **Firebase Firestore** for database management.
- **Firebase Authentication** for secure user login and role-based access control.
- **Firebase Functions** for backend logic and automation.
- **GitHub Actions** for continuous deployment.
